# Madhav Ecommerce Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze and track online sales performance across India.

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## Features
- Sales Analysis
- Profit Analysis
- Customer Analysis
- State-wise Performance Tracking
- Category-wise Analysis
- Interactive Filters and Slicers

## Dashboard Visualizations
- KPI Cards
- Bar Chart
- Pie Chart
- Donut Chart
- Line Chart
- Map Visualization
- Slicers

## Key Learnings
- Created an interactive dashboard to track and analyze online sales data.
- Used filters and slicers for dynamic reporting.
- Performed data cleaning and data modeling.
- Created DAX measures for business calculations.
- Built various visualizations to generate business insights.

## Project Outcome
The dashboard helps identify top-performing states, customers, and product categories, enabling better business decisions.
